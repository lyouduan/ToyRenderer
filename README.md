# Toy Renderer on DirectX 12

A toy  render written in C++ using DirectX 12.





## Basis

- D3D12 Memory  Management
  - Buddy Allocator
- D3D12 Descriptor Management
- Dear ImGUI
- Assimp



## Feature

- PBR and  IBL
